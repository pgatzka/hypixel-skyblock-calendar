# Setup Manifest — exactly what is project-specific

This template is **frozen** except for the items below. `/setup` fills these in a
fresh instance; everything not listed here is invariant and must not be edited
per project. Completeness is enforced: any unfilled `<<<FILL>>>` makes
`scripts/check-config.ps1` (and CI) fail.

## The per-project surface (the ONLY things /setup fills)

| # | File | What to fill | Mandatory? |
|---|------|--------------|------------|
| 1 | `.claude/gates.json` | The 5 gate commands: `format`, `lint`, `typecheck`, `test`, `build`. Each = a command string, or `""` to skip that gate. | **Yes** — this is the keystone. |
| 2 | `CLAUDE.md` → *Project overview* | 2–3 sentences on what the project is. | Yes |
| 3 | `CLAUDE.md` → *Conventions* | Languages, naming, structure, error handling, test style, commit format. | Yes |
| 4 | `.github/workflows/ci.yml` → *toolchain block* | The one per-project step: language/runtime setup + dependency install (e.g. `setup-node`+`npm ci`). Delete if none needed. | Yes (or delete) |
| 5 | `CLAUDE.md` → *Gotchas* | Seed empty; grows over time. | No (starts empty) |
| 6 | `.claude/settings.local.json` | Generated by /setup from gates.json: `permissions.allow` for the gate commands so they run without prompting. Git-ignored. | No (convenience) |

## Frozen (never edit per project)
`.claude/skills/*`, `.claude/agents/*`, `.claude/hooks/*`, `.claude/settings.json`
(permissions + hook wiring), `.github/ISSUE_TEMPLATE/`, `.github/pull_request_template.md`,
the CI gate-running logic, `docs/decisions/0000-template.md`, `scripts/check-config.{sh,ps1}`. Hook scripts ship in both `.sh` and `.ps1`; which one is wired is a documented option (default `.sh`), not a per-project fill.

## Contract
1. `/setup` replaces every `<<<FILL: ...>>>` sentinel — nothing else.
2. `""` in gates.json is a deliberate "skip"; a remaining `<<<FILL>>>` is "unconfigured."
3. `scripts/check-config.ps1` must pass (zero sentinels) before the instance is considered set up. CI runs it first and fails otherwise.
